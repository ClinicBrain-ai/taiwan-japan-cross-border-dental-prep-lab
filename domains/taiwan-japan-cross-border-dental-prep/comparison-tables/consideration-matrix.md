# Consideration Matrix

This matrix asks planning-preparation questions only. It must not conclude which country, clinic, dentist, or treatment path is better.

| Area | Questions if staying in current country | Questions if seeking care in home country |
| --- | --- | --- |
| Urgency | What should be checked locally before ordinary planning continues? | What urgent-evaluation result or professional note should be summarized? |
| Symptoms | What symptom timeline and changes should be clarified locally? | What symptom history should be prepared for review? |
| Records | Which records can be requested from the current clinic? | Which records would support continuity-of-care review? |
| Treatment stages | What stages did the current dentist describe? | What stage-related questions should be prepared for review? |
| Follow-up care | What follow-up schedule, checks, or temporary steps were described? | What follow-up questions should be asked for continuity across countries? |
| Cost | What items, exclusions, or payment timing need clarification? | What cost-estimate questions should be prepared without assuming an outcome? |
| Insurance | What coverage, reimbursement, or self-pay uncertainty remains? | What coverage questions should be prepared for home-country review? |
| Language | Which original terms should be preserved exactly? | Which terms need translation or plain-language organization? |
| Timeline | What scheduling, work, school, visa, or travel constraints exist? | What timeline questions should be reviewed by a dentist or medical professional? |
