# Language Policy

Use decision-preparation language throughout this domain module.

Preferred phrases:

- "considering whether to organize information for possible care"
- "planning preparation"
- "expert-review packet preparation"
- "cross-border dental care planning information organization"
- "bilingual expert-review packet preparation"
- "questions for local dentist review"
- "questions for home-country dentist review"
- "records for continuity of care"
- "requires dentist or medical professional evaluation"

Avoid phrases that imply the project chooses a clinical or travel path for the user.

Do not say or imply:

- The user should return home
- The user should stay
- The user should wait
- The user should travel
- The user should fly
- A treatment is the correct option
- Taiwan is better, cheaper, faster, safer, or more appropriate
- Japan is better, cheaper, faster, safer, or more appropriate
- A dentist is right or wrong

## Dentist statements and user interpretation

Always separate:

- Original wording
- Translation or plain-language organization
- User interpretation
- Uncertainty
- Questions to ask dentists

This separation keeps the packet useful without turning it into diagnosis, treatment recommendation, formal second opinion, or teledentistry.

## Image-related language

Allowed: "X-rays," "panoramic X-rays," "local dental radiographs," "CBCT / DICOM if already available," and "intraoral photos if already available" as records or attachments.

Not allowed: any statement that describes what an image shows, means, confirms, rules out, or suggests.

## Urgent warning language

Use this notice when urgent warning signs are present:

This situation should be checked locally first. This project cannot decide whether it is safe to wait, travel, or return home. Please seek local urgent dental or medical evaluation before continuing cross-border planning.
