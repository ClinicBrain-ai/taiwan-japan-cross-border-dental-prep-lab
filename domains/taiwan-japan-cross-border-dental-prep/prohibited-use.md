# Prohibited Use

This module must not become a clinical product, dental advice app, triage surface, travel-safety surface, second-opinion service, cost-guarantee tool, or country-comparison tool.

It must not provide:

- Diagnosis
- Treatment recommendation
- Image interpretation
- Medication advice
- Emergency care
- Emergency triage
- Waiting-safety judgment
- Travel-safety judgment
- Advice about whether flying is safe
- Advice about whether returning home is safe
- Teledentistry
- Formal second opinion
- Guarantee of cost, insurance, quality, speed, or outcome
- Judgment that Taiwan or Japan is better, cheaper, faster, safer, or more appropriate
- Claim that a dentist is right or wrong

It must not introduce:

- Real patient data
- Backend services
- API calls
- Authentication
- Database storage
- Analytics
- File upload
- Payment
- Email collection

## Language that is not allowed

Do not write language that implies the system decides whether a user should stay, wait, travel, fly, return home, or choose a treatment.

Allowed phrasing includes "considering whether to organize information for possible care," "planning preparation," and "expert-review packet preparation."
