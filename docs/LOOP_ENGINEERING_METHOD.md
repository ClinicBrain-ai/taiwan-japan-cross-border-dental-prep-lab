# Loop Engineering Method

The workflow is human-led, ChatGPT-assisted, and Codex-implemented.

The learning loop is: Collect → Separate → Scaffold → Bound → Artifact → Review → Learn.

- Collect: gather synthetic context, goals, constraints, questions, and known facts.
- Separate: distinguish facts, assumptions, user concerns, logistics, and clinical unknowns.
- Scaffold: place information into a repeatable expert-review packet structure.
- Bound: label what the system must not answer.
- Artifact: produce a static, reviewable preparation packet.
- Review: inspect the artifact against safety, privacy, language, and architecture constraints.
- Learn: improve the system design without expanding into clinical decision making.

This loop treats AI as an organizing layer. It can improve clarity, completeness, and review readiness, but it must not choose treatment, interpret images, assess emergency status, or decide whether waiting or travel is safe.
