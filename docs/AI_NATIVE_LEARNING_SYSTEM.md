# AI-Native Learning System

AI-native means the prototype is designed around inspectable human-AI work loops rather than a traditional form or clinical recommendation engine.

The human sets the learning goal and professional boundary. ChatGPT helps reason through structure, language, and review criteria. Codex implements local code, documentation, and tests that can be inspected in Git.

The system is AI-native but not autonomous. It should organize information into expert-review artifacts, surface missing context, and preserve uncertainty. It should not make dental decisions, rank treatments, judge dentists, or tell users what is safe.

The public framing should read as professional learning system design. It should not read as dental advice, patient triage, teledentistry, or a second-opinion product.
