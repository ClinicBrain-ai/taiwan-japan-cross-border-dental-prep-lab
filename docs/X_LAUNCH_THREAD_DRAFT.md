# X Launch Thread Draft

1. I built a new local-first prototype with ChatGPT Pro + Codex:

Taiwan-Japan Cross-Border Dental Prep Lab.

It helps Taiwanese people in Japan and Japanese people in Taiwan organize dental-care planning information into expert-review packets.

2. Not diagnosis.
Not treatment advice.
Not X-ray interpretation.
Not travel-safety judgment.

The frame is deliberately narrow:

decision preparation, not decision making.

3. The core idea:

AI should not decide whether you should return home for dental care.

But AI can help organize what a person already has, what is missing, and what questions should be reviewed by dentists.

4. The packet organizes:

- symptoms
- dentist statements
- patient interpretation
- quotes
- treatment plans
- missing records
- cost and insurance questions
- language concerns
- timeline constraints
- questions for local and home-country dentists

5. The output is a Markdown packet for dentist review.

It keeps original wording, translation/plain-language organization, user interpretation, uncertainty, and questions separated.

6. Built local-first.
Synthetic examples only.
No backend.
No real patient data.
No clinical decision support.

7. The first domain is Taiwan-Japan cross-border dental planning.

The larger work is AI-native professional learning systems for high-trust domains.

8. The part I care about most:

AI can make expert review easier without pretending to be the expert.

That design space feels much more useful than turning every serious decision into an answer box.
