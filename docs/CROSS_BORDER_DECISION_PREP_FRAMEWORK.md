# Cross-Border Decision Prep Framework

The framework organizes non-clinical planning information for a dentist review packet.

Useful categories:

- User goals and visit purpose
- Known facts from synthetic context
- Timeline and location constraints
- Taiwan-Japan language and coordination needs
- Questions for a dentist
- Documents or records to ask a dentist about
- Unknowns and assumptions
- Safety boundaries and urgent-evaluation reminders

The framework must not answer clinical questions. It can separate and label them so a dentist can evaluate them.

Cross-border planning can include travel timing, availability, communication, language, and record-transfer questions. The prototype must not decide whether the user should fly, wait, return to Taiwan, return to Japan, or seek a particular treatment.
