# Portfolio Positioning

This project should be presented as AI-native professional learning system design.

It demonstrates:

- Human-led problem framing
- ChatGPT-assisted structure and language design
- Codex-implemented local code, docs, and tests
- Synthetic-first case design
- Static architecture that avoids patient data and external services
- Explicit safety boundaries around clinical decision making
- Reviewable artifacts rather than opaque recommendations

It should not be positioned as dental advice, a clinical product, a patient-facing triage service, a travel-safety service, teledentistry, a second opinion, an insurance product, or a cost guarantee tool.

The strongest portfolio claim is disciplined boundary design: the prototype shows how AI can help organize professional preparation work while preserving the authority of licensed experts.
